## Overview

A comprehensive SaaS theme project that enables users to customize every detail of their website, including translations, icons, fonts, button colors, headers, and more. The project focuses on providing maximum flexibility while maintaining performance and user experience.

## Key Features

- Complete website customization capabilities
- Dynamic attribute components
- Country-specific payment methods
- Performance optimized with code splitting and lazy loading
- Caching static content for improved load times
- Achieved consistent A/B scores on GTmetrix

## Technical Implementation

<br />

### Frontend Stack
- NextJS as the core framework
- Redux with Redux-persist for state management
- Styled-components and SASS for styling
- GraphQL for data fetching

### Cloud Services
- AWS S3 for storage
- AWS Map integration
- Custom payment gateway integrations

## Performance Optimizations

- Implemented code splitting for optimized bundle sizes
- Utilized lazy loading for improved initial load times
- Integrated caching mechanisms for static content
- Reduced page requests through optimized asset loading

## Achievements

- Consistently achieved A score on GTmetrix
- Significantly improved page loading speeds
- Reduced overall data transfer size
- Enhanced user experience through optimized performance
